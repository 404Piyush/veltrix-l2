## Bridge App Details

Current frontend features:

- Wallet connect.
- Sepolia L1 balance load.
- Veltrix L2 balance load.
- ETH deposit through `OptimismPortalProxy`.
- ETH withdrawal initiation through `L2ToL1MessagePasser`.
- Recent bridge activity list.
- Contract copy panel.
- Withdrawal lifecycle panel.

Bridge config currently uses:

```text
L1 chain: Sepolia
L2 chain ID: 42069
L2 chain hex: 0xa455
L2 RPC: http://localhost:9546
OptimismPortalProxy: 0x9d6954E55297f9ae78e5c0dc2353c18b31aeA0b3
L2ToL1MessagePasser: 0x4200000000000000000000000000000000000016
```

For VPS/domain deployment, replace local RPC with an HTTPS public RPC:

```bash
VITE_L2_RPC_URL=https://rpc.yourdomain.com
VITE_L2_EXPLORER_URL=https://explorer.yourdomain.com/tx/
```
